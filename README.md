تنفيذ المخزن الدائري (Circular Buffer Implementation)
هذا المشروع هو تنفيذ لهيكل بيانات المخزن الدائري بلغة C.
وصف المشروع
يقوم البرنامج بتنفيذ مخزن بيانات دائري يعتمد على المصفوفات والمؤشرات بشكل يدوي، حيث يتم التحكم في تدفق البيانات (إدخال وإخراج) باستخدام مؤشري الرأس (Head) والذيل (Tail).
الميزات البرمجية
إدارة الذاكرة: العودة إلى بداية المصفوفة عند الوصول لنهايتها باستخدام معامل الباقي %.
الحماية والتحقق: البرنامج يكتشف ويحذر عند حدوث امتلاء (Overflow) أو فراغ (Underflow).
معالجة النصوص: يستقبل البرنامج الاسم، يضيف اللاحقة "CE-ESY"، ثم يخزنها ويعرضها بترتيب صحيح.
